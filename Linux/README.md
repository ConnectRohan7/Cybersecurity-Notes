# Practical Scripts

### Bash Scripts

- `aboutme.sh` — Displays basic user information.
- `backup.sh` — Creates a timestamped backup of a directory.
- `check_file.sh` — Checks whether a file exists.
- `check_name.sh` — Demonstrates conditional statements.
- `count_lines.sh` — Counts the number of lines in a text file.
- `date.sh` — Displays the current date and time.
- `directory.sh` — Creates and manages directories.
- `grade.sh` — Demonstrates conditional grading logic.
- `hello.sh` — Simple introductory Bash script.
- `intro.sh` — Bash scripting practice.
- `process_logger.sh` — Saves all running processes to a text file.
- `soc_intro.sh` — Introductory SOC-related Bash practice.
- `user_info.sh` — Displays information about the current user.
- `variables.sh` — Demonstrates the use of variables in Bash.

### Python Script

- `python-http.py` — Sends an HTTP GET request using Python's `requests` library, displays the HTTP status code, response headers, and the first 300 characters of the webpage.
