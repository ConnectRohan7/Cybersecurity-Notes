# OSI Model (Open Systems Interconnection)

## Objective

Understand the 7-layer OSI model, how data travels across a network, and the purpose of each layer.

---

## OSI Layers

### Layer 7 – Application
- Provides network services to user applications.
- Examples: HTTP, HTTPS, DNS, FTP, SSH.

### Layer 6 – Presentation
- Encrypts, compresses, and formats data.
- Example: TLS/SSL.

### Layer 5 – Session
- Establishes, manages, and terminates communication sessions.

### Layer 4 – Transport
- Provides end-to-end communication.
- Protocols: TCP, UDP.
- Data Unit: Segment (TCP) / Datagram (UDP).

### Layer 3 – Network
- Routes packets between different networks.
- Uses IP addresses.
- Device: Router.
- Data Unit: Packet.

### Layer 2 – Data Link
- Delivers frames within the same local network (LAN).
- Uses MAC addresses.
- Device: Switch.
- Data Unit: Frame.

### Layer 1 – Physical
- Transmits bits as electrical signals, light, or radio waves.
- Devices: Hub, Repeater, Ethernet Cable.
- Data Unit: Bits.

---

## Data Units

| Layer | Data Unit |
|--------|-----------|
| Application | Data |
| Presentation | Data |
| Session | Data |
| Transport | Segment / Datagram |
| Network | Packet |
| Data Link | Frame |
| Physical | Bits |
